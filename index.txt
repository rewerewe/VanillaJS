프론트엔드라면 javascript 가 유일하다.

웹에서 쓸수 있는 유일한 언어

1.3
ES6 
ECMAScript = Specification 설명문이다. 
javascript는 집중화 되어 있어서 누군가 업데이트하면 모든 브라우저에서 작동함.
많은 사람들에게 검증받은 설명문/체계 매뉴얼 같은 것. 
크롬, 익스플로러 등 브라우저는 이 Specification을 받아서 자기 방식으로 실행함.
각자의 방식으로 같은 결과를 이루려고 노력함. 
ES6은 버전을 말함. 각각이 뭔지를 몰라도 되고 
컨셉을 생각하고, 그 컨셉을 배우면 됨.
버전이랑 어떤걸 배워야할지 걱정하지말고 VanillaJS 를 배우면 됨. 
VanillaJS = JavaScript without Library

1.4 
VanillaJS
바닐라 자바스크립트는 브라우저를 통해 제공된 JavsScript라고 보면 됨.
know = JavaScript Library, Framework = react, Jquery 
don't know = VanillaJS
본질을 모르고 사용하는 사람들이 많은 언어. 
VanillaJS는 대체할 수 있는 것이 아닌 실질적인 언어.
다른 언어보다 빠르고 정확함. 
다른 언어, Framework 로 넘어갈 때에도 문제 없어진다.
모든 웹의 베이스가 되는 JavaScript를 배워두는 게 좋다. 
예쁘고 우아하진 않지만 웹 상에서 본 모든 것은 이 언어로 구성되어 있다.
javascript가 강력한 이유.
